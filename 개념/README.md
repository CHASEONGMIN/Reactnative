## 리액트 네이티브 동작 방식

### 어떻게 자바스크립트 코드를 네이티브 코드로 변경을 하는가?

- 자바스크립트코드와 네이티브 계층과의 통신을 할 수 있게 해주는 브릿지가 존재함.

### 가상 DOM

데이터 변화시 FLOW

1. 데이터 변화 발생
2. 변화된 데이터를 이용하여 가상 DOM 그림
3. 가상 DOM과 실제 DOM을 비교하여 차이점 확인
4. 차이점이 있는 부분만 실제 DOM에 적용/
